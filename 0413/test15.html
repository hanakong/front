함수 연습

    0         1         2         3
    0123456789012345678901234567890
    ABCDEFGHIJKLMNOPQRSTUVWXYZabcde

length : 객체 or 문자 or 문자열의 길이를 구하는 함수

substring(a,b) : 문자열 발췌(a에서부터 b 앞까지)

slice(a,b) : substring과 같음

substr() : substring과 거의 같음//현재는 거의 안씀

replace(a,b) : a를 b로 치환

replaceAll(a,b) : a를 b로 모두 치환

toUpperCase() : 대문자로 변환

toLowerCase() : 소문자로 변환

concat(a,b) : a와 b문자열 결합

trim() : 문자열의 앞뒤(양쪽) 공백을 무시

trimStart() : 문자열의 앞쪽 공백을 무시

trimEnd() : 문자열의 뒤쪽 공백을 무시

replaceAll을 통해 공백 모두 제거

charAt() : 하나의 문자만 발췌

charCodeAt(a) : a문자의 아스키코드를 출력

split() : 특정문자를 기준으로 잘라낸다. 이때 잘라낸 값들은 배열변수에 기억된다.

indexOf() : 문자(열) 검색시에 사용 : 있으면 위치값을, 없으면 -1을 반환

lastIndexOf() : 문자(열) 검색시에 사용(뒤에서부터 검색) : 있으면 위치값을, 없으면 -1을 반환

search() : 문자(열) 검색시에 사용 | 있으면 위치값을, 없으면 -1을 반환

match() : 문자열 일치여부.. | 있으면 문자열을 그대로 반환, 없으면 공백 반환

includes() : 문자열 포함여부... | 포함하고 있으면 true, 없으면 false
